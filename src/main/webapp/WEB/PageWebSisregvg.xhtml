Sobre Sisregvg

´

´

´

Los trabajadores del IESP Valle Grande solicitan un importe de dinero, para poder comprar materiales o recursos de estudios y a la vez para la empresa. Ahora que estamos en pandemia no se podra ir al establecimiento y recibir el documento físico, con el importe solicitado. Por este motivo se ha creado el sistema de software SISREGVG con la finalidad de poder ayudar a gestionar y controlar los documentos de egresos asi tambien generando los documentos via web desde la comodidad de tu casa.

Beneficios

TIEMPO

Ahorro en tiempo desplazarce para poder conseguir las boletas emitidas por el instituto con el proyecto de software todo sera virtual .

DINERO

Ahorro en dinero ya no se deberia comprar hojas para poder obtener las boletas en fisico,con el proyecto se suplanta por boletas electrónicas.

SEGURIDAD

Seguridad de los datos ingresados, sin pérdidas ni fallos, usando un correcto estándar.

Funcionalidades

El sistema Sisregvg cuenta con :

Registro del Personal

Provee un control del personal registrado.

Registro de Liquidación

Provee un control de egresos realizados por el personal.

Vale Provisional

Provee un control de solicitud emitida por el personal de trabajo..

Registro de Reembolso

Provee un control del monto gastado del personal y el monto a devolver
